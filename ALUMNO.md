## ¿Qué es Git?
Git es un sistema de control de versiones que permite guardar cambios en un proyecto y regresar a versiones anteriores cuando es necesario.

## ¿Por qué usar Git?
Permite trabajar de forma ordenada, mantener historial de cambios y evitar perder información al modificar archivos.

## Conceptos principales
- Repositorio: carpeta donde se guarda el proyecto y su historial.
- Commit: registro de cambios realizados.
- Rama: línea de desarrollo independiente.

## Comandos utilizados
git init  
git status  
git add  
git commit  
git push  
git pull  

## Reflexión
Git facilita el control del trabajo y permite organizar mejor los cambios dentro de un proyecto.
